# ГеоБлог (Flask)

Веб-приложение блога в тематике геодезии/картографии/ГИС.

## Функционал
- Регистрация и вход
- Создание постов (публичные/приватные/скрытые)
- Подписки и лента по подпискам
- Комментарии
- Теги и фильтрация по тегам
- Поиск по постам
- Лайки
- Профиль пользователя
- Изменение темы

## Установка и запуск

Для запуска приложения необходимо предварительно установить Python 3. Далее требуется клонировать репозиторий на локальный компьютер, создать и активировать виртуальное окружение, установить зависимости и запустить сервер разработки Flask.

Выполните следующие команды в терминале:

```bash
git clone https://github.com/psodifjro/geoblog.git
cd geoblog
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py

После запуска приложение доступно по адресу:
http://127.0.0.1:5000
